Exercício de HTML

Exercício de HTML do módulo 03 de Engenheiro Front-End da EBAC.

Criar um arquivo HTML que contenha um formulário de cadastro, com os campos nome, e mail e senha, e a existência de um botão para realizar o cadastro.

Em seguida, enviar esse arquivo para o Github em uma branch chamada exercício_html e enviar o link pelo espaço dedicado a esse fim.
